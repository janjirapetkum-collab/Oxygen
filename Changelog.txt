[COMBAT UPGRADES]

Global Localization: Critical Roblox functions (Camera, WorldToViewport, etc.) are now cached. This significantly reduces CPU usage and FPS drops on mobile devices during combat.
Signal Cleaner: Implemented SafeTag logic to prevent "Instance Spam." The ESP now checks for existing labels instead of creating new ones every frame, preventing memory leaks.
Heartbeat Sync: Combat calculations moved to the Heartbeat signal to stay synced with the game's physics engine rather than the display framerate.
ipairs Optimization: All player and NPC loops now use ipairs for faster table iteration on Luau VM.
[BYPASS & SECURITY]
V3.5 Bypass Engine: Expanded "ExecutorDetection" blocking. The script now monitors for specific remote naming patterns (e.g., "AdCheck", "Detection") used by modern anticheats.
Metatable Recovery: Added an auto-recovery system that re-injects hooks upon character death/respawn to prevent the game from clearing the script's memory modifications.
Checkcaller Protection: All hooks now use checkcaller() verification to ensure Oxygen's internal scripts never conflict with the game's hooked functions.

Fixed an issue where Silent Aim would cause "Invalid Ray" errors on certain mobile executors.
